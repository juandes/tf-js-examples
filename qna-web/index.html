TensorFlow.js Pre-trained "QnA" BERT model

Introduction

This application hosts TensorFlow.js' pre-trained Question and Answer model, and it attempts to answer the question using a given passage. To use it, write a passage (any!) in the input area below and a question. Then, click the "answer!" button to answer the question using the given passage as a reference. You could also click any of the test buttons to load a pre-defined input text.
